# Project 1

팀명: 이가네  
팀장: 이영민

1. 프로젝트 주제
영어 단어를 사용자가 등록 및 수정하여 서버에 데이터를 반영하고, 서버의 단어를 가져와 간단한 TEST를 볼 수 있는 웹 어플리케이션 제작

2. 주제 선정 이유
- HTML 구조, CSS styling과 JavaScript의 이벤트를 적극 활용할 수 있다.
- 서버의 데이터를 가져오고 클라이언트에서 입력한 정보를 반영하여 CRUD 하는 것을 연습할 수 있다.
- 모던 레이아웃 기법인 Grid를 활용하여 화면을 구성하며, Sass를 통해 반응형 웹 구현에 도전해본다.

3. 그림
  - TAB 1 (WORDS) : 영단어를 입력하여 조회하고, 서버데이터에 있으면 내용을 보거나 수정, 서버데이터에 없으면 새로 등록할 수 있으며 각 요소, 그리고 전체 목록을 삭제할 수 있다.
  - TAB 2 (TEST GAME) : 등록한 영단어 목록에서 10개의 단어를 꺼내와 해당 단어의 뜻을 입력하여 맞추는 게임을 수행하고, 게임 후에는 점수를 확인할 수 있다.
  - mobile : 특정 device-width 내에서 레이아웃의 콤포넌트 순서가 바뀌고 약간의 화면 변화가 있다.
